Curriculum

Educación

html5

Platzi 2020-2022

Realicé un emocionante curso de HTML en el que adquirí una base sólida para el desarrollo de páginas web. Durante este curso, tuve la oportunidad de sumergirme en el fascinante mundo del lenguaje de marcado utilizado para crear y estructurar contenido web.

Aprendí los conceptos básicos de HTML, desde etiquetas y elementos hasta atributos y sintaxis. Fue muy emocionante explorar las diferentes versiones de HTML, centrándome especialmente en HTML5, la versión más actual y ampliamente utilizada en la actualidad. Descubrí cómo estructurar una página HTML, utilizando etiquetas para definir encabezados, párrafos, listas, enlaces, imágenes y otros elementos comunes.

Curso de css3

Platzi: 2020-2022

Durante el curso, exploré los fundamentos de CSS y descubrí cómo utilizarlo para dar estilo a los elementos HTML de manera efectiva. A medida que avanzaba, me di cuenta de que CSS es mucho más que solo cambiar colores y tamaños de fuentes. Es un lenguaje poderoso que permite crear diseños atractivos y personalizados.

Además, descubrí cómo utilizar propiedades CSS para controlar el tamaño, el color, la posición y otros aspectos de los elementos. Experimenté con diferentes propiedades y sus valores para crear diseños únicos. Por ejemplo, pude jugar con los márgenes, el relleno y los bordes para lograr una presentación visualmente atractiva.

Curso de Javascript

Platzi 2020-2022

Durante mis cursos de JavaScript, aprendí los conceptos fundamentales que sustentan este lenguaje de programación. Comencé con los conceptos básicos, como variables, tipos de datos y operadores. A medida que avanzaba, adquirí un conocimiento más profundo sobre las estructuras de control, como bucles y condicionales, que me permitieron crear lógica y tomar decisiones en mis programas.

Además, aprendí sobre la manipulación del DOM (Modelo de Objetos del Documento) utilizando JavaScript. Esta habilidad me permitió interactuar con los elementos de una página web, modificar su contenido y responder a eventos del usuario. Pude crear interactividad en mis proyectos y hacer que mis sitios web fueran más dinámicos.

Curso de React.js

Platzi 2020-2022

Tuve la increíble oportunidad de realizar unos cursos de React.js y quiero compartir mi experiencia. Durante el curso, pude sumergirme en el fascinante mundo de React.js y descubrir su poder para crear interfaces de usuario dinámicas y reutilizables.

Una de las cosas que más me gustó de React.js es su enfoque en el manejo del estado. Aprendí a utilizar el estado para representar y controlar los datos cambiantes de mi aplicación. La función setState se convirtió en una herramienta poderosa para actualizar el estado y provocar la actualización automática de los componentes afectados.

Experiencia de trabajo

Tabla de tareas

TalenLogy 2023

Como desarrollador, me enfrenté al desafío de crear una tabla de tareas utilizando HTML, CSS y JavaScript. Esta tabla de tareas me permite organizar mis actividades diarias de manera efectiva.

Luego, pasé a diseñar el aspecto visual de la tabla utilizando CSS. Agregué estilos para los encabezados de columna, las filas de tareas y los diferentes estados de las tareas, como "pendiente", "en progreso" y "completada". Además, agregué estilos de resaltado para las tareas seleccionadas.

Para interactuar con la tabla de tareas, agregué eventos a los elementos HTML utilizando JavaScript. Por ejemplo, cuando hago clic en el botón "Agregar tarea", se ejecuta una función que recopila los datos ingresados por el usuario, crea una nueva fila en la tabla y actualiza los datos almacenados en localStorage. De manera similar, cuando hago clic en el botón "Eliminar tarea" en una fila, se elimina esa tarea específica de la tabla y se actualizan los datos en localStorage.
